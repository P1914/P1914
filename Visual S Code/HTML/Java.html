១. ដើម្បីលើកម្ពស់វិស័យសិក្សាធិការជាតិ តើរាជរដ្ឋភិបាល សម័យសង្គមរាស្រ្ដនិយមបានធ្វើដូចម្ដេចខ្លះ ?

១. ដើម្បីលើកកម្ពស់វិស័យសិក្សាធិការជាតិរាជរដ្ឋាភិបាល សម័យសង្គមរាស្រ្ដនិយម បានធ្វើដូចតទៅ ៖
+ អភិវឌ្ឍវិស័យ អប់រំឪ្យមានគុណភាពនិងប្រសិទ្ធិភាព ដោយបានបណ្ដុះបណ្ដាលគ្រូ បង្រៀន និងកសាងសាលាតរៀន ចាប់ពីបឋមសិក្សារហូតដល់សាកលវិទ្យាល័យ ។
+ នៅឆ្នាំ ១៩៦០ បង្កើតឪ្យមានសាកលវិទ្យាល័យភូមិន្ទភ្នំពេញ មហាវិទ្យាល័យនីតិសាស្រ្ដ មហាវិទ្យាល័យវិទ្យាសាស្រ្ដ មហាវិទ្យាល័យវេដ្ជសាស្រ្ដ...​។
+ ខាងសាសនា គឺបង្កើតឪ្យមានពុទ្ធិបឋមសិក្សា រហូតដល់ពុទ្ធិមហាវិទ្យាល័យ ។​
+ នៅឆ្នាំ ១៩៦១ បានលើក ស្ទួយវិស័យ កីឡាពោលគឺបានជួបគ្រូបង្ហាត់ ជួលគ្រូបង្វឹកមកពីបរទេសដូចជា ៖
កីឡាបាល់ទាត់ ជួលគ្រូបង្វឹកមកពីឆេកូស្លូវ៉ាគី
សម្រាប់កីឡា បាល់ទះ ជួលគ្រូបង្វឹកមក ពីសហរដ្ឋអាមេរិក
សម្រាប់កីឡា វាយសីជួលគ្រូបង្វឹកមកពីប្រទេស អូស្ដ្រាលី
បង្កើតកីឡា នៅតាមក្រសួងមន្ទីរ សាលារៀន សហគ្រាសរដ្ឋឯកជន ។
បង្កើតឪ្យមានទីលានកីឡា ផ្សេងៗ នៅគ្រប់ស្រុក ខេត្ត ក្រុង ។

២. ដើម្បីដោះស្រាយបញ្ហាសុខមាលភាព និងជីវភាពរស់នៅរបស់ប្រជាជន តើរាជរដ្ឋាភិបាលសម័យសង្គមរាស្រ្ដនិយមបានធ្វើដូចម្ដេចខ្លះ ?

២. រាជរដ្ឋាភិបាលសម័យសរាស្រ្ដនិយម បានធ្វើដូចតទៅ ៖
ចាត់ឪ្យមានការកសាងមន្ទីរពេទ្យគ្រប់ខេត្ត ក្រុង ដើម្បីការពារនិងព្យាបាលជំងឺឪ្យប្រជារាស្រ្ដ ។
​ អភិវឌ្ឍជនបទដូចជា ការណែនាំអំពីក្បូនអនាម័យតាមលំនៅដ្ឋានការចូលរួមសម្អាត ភូមិស្ថាន ការសិក្សាពីវិធានការណ៍ការពារ និងព្យាបាលជំងឺដល់ប្រជាជនទូទាំងប្រទេស ។ ការឈូសឆាយដីបង្កើតភូមិនិងសង់លំនៅដ្ឋានជូនប្រជាជនក្រីក្រ ។ ជីកណ្ដូងទឹក សាងសង់ស្រះប្រឡាយទឹក ... ដើម្បីឆ្លើយតបទៅនឹងើតម្រូវការរបស់ប្រជាជន ។ នៅឆ្នាំ ១៩៦២ បង្កើតឪ្យមានមជ្ឈមណ្ឌូលអប់រំស្រ្ដីពេស្យនៅទួលគោក ។ ផ្ដល់សិទ្ធិឪ្យតស្រ្ដីឈរឈ្មោះបោះឆ្នោត និងមានសិទ្ធិបោះឆ្នោត ។

៣. តើព្រះបាទនរោត្តម សីហនុមានស្នាព្រះហស្ថសំខាន់ៗ អ្វីខ្លះនៅសម័យសង្គមរាស្រ្ដនិយម ?

៣. ព្រះនរោត្តម សីហនុមានស្នាព្រះហស្ថសំខាន់ៗដូចតទៅ ៖ ទាមទារឯករាជ្យពីពួកអាណានិគមបារាំង ជូនជាតិនិងប្រជាជនខ្មែរ ។ បង្កើតនិងធ្វើឪ្យមានរីកការចម្រើន នៅសម័យសង្គមរាស្រ្ដនិយម ។ ទាមទារប្រាសាទព្រះវិហារពីថៃ ដែលបានចូលទៅកាន់កាប់ដោយខុកច្បាប់នៅឆ្នាំ ១៩៥៤ បង្កើតសមិទ្ធិផលលនៃ ហេដ្ឋារចនា សម័្ពន្ធសង្គម សម័យសង្គមរាស្រ្ដនិយម ។

៤. ក្រោយពីឈប់ទទូលជំនួយពីដសហរដ្ឋអាមេរិក តើព្រះអង្គបានធ្វើដូចម្ដេច ?

៤. ក្រោយពីឈប់ទទួលជំនួយពីសហរដ្ឋអាមេរិកព្រះអង្គបានធ្វើដូចជា ៖ ទទួលយកជំនួយសម្ភារៈយោធាមួយចំនួន សម្រាប់កងចខេមភូមិន្ទពីសំណាក់ប្រទេសចិន ធ្វើជាតូបនីយកម្មក្នុងវិស័យនីហរ័ណ អាណរ័ណ ផ្នែកសេដ្ឋកិច្ច ។ បិទទ្វារធនាគារឯកជនខ្មែរទាំងឡាយចោល ។
